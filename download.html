Homedownloadobs chatbox



how to download:

  1. download the zip file: download file here
    DO NOT UNZIP IT.
  2. go to your extensions tab, for
    - Chrome: chrome://extensions
    - Edge: edge://extensions
    ...
  3. activate developer mode.
  4. and then drag the zip file onto the extensions page.
  5. go to any twitch page and it will work :D

how to use it:

  1. by clicking the in the top right side of your browser bar you can open the extension menue.
  2. by clicking on any image it will paste the "code" in your clipboard.
    a code looks like this: ::GE:: or :::Noice:::
  3. the :: on either side of the code (like this ::[emote name here]::) means that the emote is a image and
    the ::: on either side of the code (like this :::[emote name here]:::) means that the emote is a gif.
  4. use ctrl+V to paste the code in the chat. as soon as you send the message it will interpret the message and replace it with the emote.
  5. go to any twitch page, type in chat and it will work :D
    (only those who have the extension installed can see the emotes!)
